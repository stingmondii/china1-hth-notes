# china1-hth-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容独立，不依赖于特定域名或网站。

## 项目简介

这是一个用于存放 HTML 页面归档资料的仓库。所归档的页面均为独立 HTML 文件，无需额外后端支持，可直接在浏览器中打开查看。

## 目录说明

- `pages/` — 存放主要的 HTML 页面文件
- `assets/` — 存放页面所需的资源文件（如样式、图片、脚本等）
- `archive/` — 存放历史版本或不再维护的页面

## 页面归档说明

每个 HTML 页面均为独立自包含文件，尽量使用内联样式和脚本，减少对外部资源的依赖。页面命名遵循以创建日期或主题标识的方式，便于识别和检索。

## 维护说明

- 本仓库主要进行内容归档和版本管理，不承诺定期更新。
- 欢迎通过提交 Issue 或 Pull Request 提出建议或修正。
- 页面内容的准确性、合法性由贡献者自行负责。
